Window Summary

There are two Windows, the Object Pane and the Control Window. The Object Pane shows the x and y axes, as well as any objects within the specified range. The Control Window contains the menus for controlling different parts of the Object Pane.

Object Menu

The Object Menu contains the controls to add objects to the Object Pane

There are six fields that need to be completed to add and object:
1)The velocity of the object given in arbitrary units
2)The direction of given velocity in degrees, with 0 being East, 90 being North etc.
3)The acceleration of the object in arbitrary units
4)The direction of said acceleration in degrees, with 0 being East, 90 being North etc.
5)The location x of the object
6)The location y of the object

Once the fields have been properly filled out, pressing "Add Object" will add the object to the Object Pane.

Simulation Menu

The Simulatoin Menu controls the simulation in the Object Pane

There are three buttons in the Simulation Menu:
1)The Start button starts the simulation
2)The Stop button stops the simulation
3)The Clear button clears all objects from the simulation

Window Menu

The Window Menu controls the bounds of what is visible in the Object Pane

There are four fields to be filled out to adjust the bounds:
1)The x lower bound, the lowest x value visible in the Object Pane. Default -350
2)The x upper bound, the highest x value visible in the Object Pane Default 350
3)The y lower bound, the lowest y value visible in the Object Pane. Default -350
4)The y upper bound, the highest y value visible in the Object Pane. Default 350

Once the fields have been properly filled out, pressing "Set" will set the visible window in the Object Pane
